# 📚 Algoritmos e Programação de Computadores - Resolução de Exercícios

Este repositório contém as minhas resoluções para as listas de exercícios e atividades práticas da disciplina de **Algoritmos e Programação de Computadores**, lecionada pelo professor Saulo Santos. 

Os ficheiros aqui presentes documentam a minha progressão prática, desde a construção da base lógica até à exploração de dados, refletindo a aplicação dos conceitos lecionados ao longo das 12 aulas do curso.

## 🚀 Tecnologias e Ferramentas Utilizadas
* **Linguagem:** Python
* **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook
* **Bibliotecas Principais:** Pandas (para exploração e manipulação de bases de dados)

## 📂 Estrutura do Repositório
Todo o trabalho prático foi desenvolvido em *notebooks* e está dividido da seguinte forma:

* **Aulas 1 a 6:** Entrada e saída de dados, estruturas de decisão, álgebra booleana e estruturas de repetição.
* **Aulas 7 a 9:** Estruturas de dados (Listas e Dicionários) e criação de Funções.
* **Aulas 10 a 12:** Manipulação de ficheiros (TXT, CSV, JSON) e projeto final focado na exploração de dados (Kaggle) utilizando Pandas.
